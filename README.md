## 📖 关于本软件

自动音乐播放器是一款 Windows 桌面工具，可定时自动播放音乐，支持背景图切换、倒计时、托盘隐藏等特性。

- 基于 .NET 10 开发
- 仅支持 Windows 10 / 11 x64
- 开源免费，采用 MIT 协议

---
